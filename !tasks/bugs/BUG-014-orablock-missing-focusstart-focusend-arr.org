* Intention & Purpose

Restore caret handoff between sibling blocks. After this lands: ArrowUp/Down across siblings parks the caret at end/start of the destination block; Enter at end-of-block creates a sibling below and caret enters it at offset 0. This is the prerequisite for BUG-016 (format toolbar — needs valid Lexical selection), BUG-017 (slash menu — needs caret in the host block), and BUG-022 (PageHeader cross-tab refresh isn't blocked by this but exercises the same focus discipline).

#+begin_src js
import { $getRoot } from "lexical";
export function moveCaretToStart(editor) {
  editor.update(() => {
    const r = $getRoot();
    r.getFirstChild() ? r.selectStart() : r.select();
  });
}
export function moveCaretToEnd(editor) {
  editor.update(() => {
    const r = $getRoot();
    r.getLastChild() ? r.selectEnd() : r.select();
  });
}
