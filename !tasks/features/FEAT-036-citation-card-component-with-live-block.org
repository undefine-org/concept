* Intention & Purpose
Single source of truth for rendering a Citation row. Hover triggers a popover that renders the *actual* target block via ~ConceptWeb.BlockRender.block/1~ — the citation preview IS the document. Shared between ChatPanel, AI block, SemanticPalette, LiveCitationRail.

* Scope (IN)
- Function component ~ConceptWeb.Components.CitationCard~ taking ~%Citation{}~ + ~workspace~.
- Renders: source-type icon (~hero-document-text~ for title hits, ~hero-sparkles~ for semantic), breadcrumb chevron path, 2-line snippet, score sparkline.
- Hover (mouseenter, 300ms debounce): pushes ~load_block_preview~ event with block_id; on reply, renders inline popover containing ~BlockRender.block/1~ of the live block.
- Click: navigates to ~/w/<slug>/p/<page_id>#block-<block_id>~ via ~<.link navigate=...>~.
- Colocated Lit element ~ora-citation-popover~ handles positioning + outside-click dismiss.

* Tests (5 scenarios)
1. Renders title-hit shape (no snippet, document icon).
2. Renders semantic-hit shape (snippet, sparkles icon, breadcrumb).
3. Score sparkline reflects ~score~ float (test against ~aria-valuenow~).
4. Click target href matches deep-link pattern.
5. Hover triggers ~load_block_preview~ event (use ~render_hook~).
