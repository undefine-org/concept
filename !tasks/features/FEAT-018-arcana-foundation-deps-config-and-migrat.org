* Edge Cases
- Missing pgvector: `mix ecto.migrate` will fail on Arcana's chunks migration. Recovery: ensure pgvector OS extension installed (`apt install postgresql-15-pgvector` or compile from source); migration FEAT-018::pgvector-migration provides the SQL `CREATE EXTENSION`.
- Gemini quota exhaustion at boot: `runtime.exs` only verifies key presence, not validity. Real API failures bubble up at first ingest.
- Arcana version drift: pin `~> 2.0` (allows 2.x patch updates only). Major bump will require revisiting FEAT-023 (which depends on undocumented graph schemas).
- `mix arcana.install` may fail if Igniter compatibility shifts. If so, manually copy migration templates from `deps/arcana/priv/repo/migrations` and document in README.
