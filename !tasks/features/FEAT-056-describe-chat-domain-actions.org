|----------------------+-----------+-----------+-------------|
| ~:create~            | create    | YES       | "Start a new chat conversation in the workspace." |
| ~:read~ default      | read      | YES       | "Read a chat conversation by id." (override the default similarly to Pages.Page :read.) |
| ~:my_conversations~  | read      | YES       | "List the actor's chat conversations in the workspace, most recent first." |

"Optional conversation title. Defaults to an AI-generated label." (if

* Actions to describe — Chat.Message

| Action                | Type    | Describe? | Description |
|-----------------------+---------+-----------+-------------|
| ~:for_conversation~   | read    | YES       | "List messages in a conversation in chronological order." |
| ~:create~             | create  | YES       | "Send a message in a chat conversation; an AI response is generated asynchronously." |
| ~:respond~            | update  | NO        | (worker-only) |
| ~:upsert_response~    | create  | NO        | (worker-only) |

Per-arg:
- ~:for_conversation.conversation_id~ — "Chat conversation id."
- ~:create.conversation_id~ — "Conversation to post the message into."
- ~:create.body~ (or whatever the message text attribute is — verify) — "Message body in plain text or markdown."
