* Intention & Purpose
Right-side panel for ad-hoc workspace Q&A. Every submitted question creates a Knowledge.Query row; streaming tokens arrive via PubSub; finalized answer renders with CitationCard children. Toggle button in sidebar (~⌘J~).

* Scope (IN)
- ~ConceptWeb.WorkspaceLive.ChatPanel~ LiveComponent. Subscribes to ~conversation:<id>~ topic (AshAI ~gen.chat~ broadcasts there). Streams ~:messages~ keyed by message_id.
- WorkspaceLive subscribes per-conversation; forwards via ~send_update/2~ to the component (pattern: LiveComponents don't handle ~handle_info~).
- Submit form: prompt textarea + scope dropdown (Workspace / Page / Subtree) + profile selector (defaults to ~:default~ — gemini-3-flash) + Send.
- On submit: calls ~Concept.Knowledge.ask/3~ which:
  1. Finds/creates a Conversation for this user + workspace.
  2. Creates a user Message with prompt + scope + profile attrs.
  3. Enqueues AshOban worker that runs ~AshAi.ToolLoop.stream/2~ with the profile's tool list + model.
- On token broadcast: append to streaming Message body in the panel.
- On ~:complete~ broadcast: render finalized message with CitationCard children (rank-sorted).
- On ~:failed~: error bubble with retry button.
- ⌘J toggles panel; Esc closes.
- "New chat" archives current Conversation (AshArchival) + starts a new one.
- History sidebar (deferred to FUP-008 promotion) reads Conversation list.
