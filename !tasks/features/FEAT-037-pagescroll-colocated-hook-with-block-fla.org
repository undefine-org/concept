* Intention & Purpose
Make clicking a citation feel native: navigate → scroll-to-anchor → 1.6s yellow flash on the target block.

* Scope (IN)
- Colocated hook ~PageScroll~ on the page editor root that:
  1. On mount, reads ~window.location.hash~; if ~#block-<uuid>~, ~scrollIntoView({behavior: "smooth", block: "center"})~ then toggles ~.ora-block-flash~ for 1.6s.
  2. Listens for ~hashchange~ events; same behavior.
  3. Handles ~phx:scroll_to_block~ events (server-pushed) for in-page deep-links without changing URL.
- CSS keyframe ~@keyframes ora-block-flash~ — background fades from yellow-100 to transparent.
