* Intention & Purpose
Cmd-K (Ctrl-K on Linux/Windows) opens a Petal-styled overlay command palette with: title-only ILIKE page search across the workspace, quick actions (New page, New page in current page, Sign out, Toggle archived view), and arrow-key + Enter selection. Replaces the inert search box in the sidebar.

* Context & Narrative
Title-only ILIKE keeps the implementation tiny while satisfying the "Search in MVP" decision; Postgres pg_trgm index keeps it fast.

* Scope
** In-scope
- =ConceptWeb.CommandPaletteLive= live_component embedded into =WorkspaceLive=, hidden by default; opened via =push_event "open_command_palette"= triggered by global keybinding hook =assets/js/hooks/global_keys.js=
- Global keys hook listens for Cmd-K / Ctrl-K, pushes =:open_palette= to LV
- Server action =Concept.Pages.search_pages_by_title/2= executes Ash read action =:search_titles= filtering =title ILIKE ^"%#{query}%"= AND =archived_at is_nil=, ordered by updated_at desc, limit 20
- pg_trgm index on =pages.title= via migration =CREATE INDEX pages_title_trgm_idx ON pages USING gin (title gin_trgm_ops)=
- Action items hardcoded:
  - "New page" → =Concept.Pages.create_page!=
  - "New child page" (if a page is currently open) → =create_page!= with =parent_page_id=
  - "Sign out" → push_navigate to =/sign-out=
  - "Show archived" → toggle a sidebar pref (FUP for full archive viewer; here it just adds a flag)
- Petal Modal-style overlay with input, sectioned results, keyboard nav

** Out-of-scope
- Full-text search across block content (FUP)
- Recent pages section (FUP — would require activity log)
- Page action commands (rename, archive, move) — FUP

** ITEM Search action + pg_trgm index

:CUSTOM_ID: FEAT-014-cmd-k-command-palette-title-search-and-q::search-action

* Edge Cases
- Empty query: show only Actions section + "Recent" (FUP — show empty pages section).
- Query with =% _= SQL wildcards: escape via =Postgrex= parameterization (Ash handles).
- Query of 100+ chars: truncate to 100.
- Workspace with 50k pages: pg_trgm keeps query under 50ms; limit 20 caps payload.
- Cmd-K inside Lexical editor: still triggers palette (preventDefault wins).
- Palette open while modal open: only one floating overlay at a time; later one wins.

- =mix test test/concept_web/live/command_palette_test.exs= green (6 scenarios)
