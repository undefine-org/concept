Make the system observable. Per-profile latency p50/p95, per-workspace daily Gemini token counts, ingestion job throughput, citation hit rates. Surface in ~/dev/dashboard~ and ~/admin/arcana~.

- New telemetry events emitted by Reactor steps + Query worker + Knowledge.Search.
