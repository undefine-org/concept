*Substantially smaller than originally planned.* AshAI's ~gen.chat~ already generates ~Conversation~ + ~Message~ resources (landed in FEAT-033 Wave 1). What's left for this FUP:

- History sidebar UI: lists Conversations sorted by ~last_message_at~; click → loads thread into ChatPanel via ~send_update~.

- Drag-to-block: drag a finalized assistant Message → drops as an AI block on current page (block content ~{message_id: msg.id}~ — reuses FEAT-041 substrate).
- Multi-turn retrieval: prior assistant messages' citations join the search context for the next turn (modify the ~:search_workspace~ tool to read parent message's search_trace).
