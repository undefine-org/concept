** Why this matters

The pair of bugs it would have caught:
1. Subscribing to ~Concept.PubSub~ while ~Message~ broadcasts via ~ConceptWeb.Endpoint~ \u2014 silent.
2. Pattern-matching on a raw map while ~Ash.Notifier.PubSub~ delivers ~%Phoenix.Socket.Broadcast{}~ \u2014 silent.

Both are cross-process protocol mismatches with zero compile-time signal.
Only an end-to-end test catches them. The matcher unit test pins half the
contract; this integration test pins the rest.
